# Foundations: Coding with Generative AI

_A CUNY AI Lab workshop introducing AI-assisted development through core concepts, environment setup, and Gemini CLI installation._

This workshop is the first in the CAIL Spotlight series, designed to give participants a working development environment and a conceptual grounding in how large language models generate code. The session covers LLM concepts, VS Code and command line basics, Git/GitHub workflows, and Gemini CLI setup.

**Workshop Date:** Spring 2026
**Duration:** 90 minutes
**Format:** CAIL Spotlight Workshop #1
**Facilitators:** Stefano Morello, Zach Muhlbauer, Parisa Setayesh, and Stephen Zweibel (CUNY AI Lab)

_This workshop was originally conducted as part of the CAIL Spotlight series at the CUNY Graduate Center._

## About the CUNY AI Lab

The **CUNY AI Lab** is a faculty and staff-led incubator for experimentation with AI by CUNY researchers across disciplines, strengthening CUNY's position at the leading edge of technological innovation. Located at the CUNY Graduate Center and led by experienced scholars and technologists who have built widely used open publishing platforms, the Lab fosters momentum for AI experimentation while foregrounding ethical practices and values that reinforce CUNY's mission of accessible, equitable, and transformative education.

This workshop is part of the CUNY AI Lab's commitment to developing critical AI literacy among faculty, staff, and students across the university system.

## Overview

Through three 30-minute segments, participants move from understanding what LLMs are and how they generate code, to setting up a development environment with VS Code and Git, to installing and authenticating with Gemini CLI. The session prepares participants for the follow-up workshop on building prototypes with AI coding agents.

## Learning Objectives

- Understand how large language models generate code through probabilistic pattern matching
- Set up a development environment with VS Code and the integrated terminal
- Navigate the command line and perform basic file operations
- Initialize a Git repository and push code to GitHub
- Install and authenticate with Gemini CLI

## Workshop Structure

### Part 1: Concepts & Context (30 min)
1. **Icebreaker** - "Where does coding live in your imagination?"
2. **Demo** - Live vibe-coding walkthrough with Gemini CLI
3. **What is an LLM?** - Probabilistic pattern engine for text
4. **How LLMs Produce Answers** - Tokenize, embed, transform via attention, predict
5. **How LLMs Generate Code** - Code as another language pattern
6. **Strengths & Limitations** - Autocomplete and intent inference vs. silent errors and hallucinated functions

### Part 2: Development Environment (30 min)
7. **What is VS Code?** - Free code editor with built-in terminal, Git support, and extensions
8. **Setting Up** - Download, install, open integrated terminal
9. **Command Line Basics** - `pwd`, `mkdir`, `cd`, `touch`, `ls`, `mv`
10. **What is Git?** - Version control (local) vs. GitHub (cloud)
11. **Git Basics** - `git init`, `git add`, `git commit`, `git status`, `git log`, `git push`
12. **GitHub CLI** - `gh auth login` and push workflow

### Part 3: Installation & Setup (30 min)
13. **Installing Gemini CLI** - Homebrew (macOS) or npm (Windows)
14. **Login Step** - Google authentication, free tier limits
15. **Troubleshooting** - Group troubleshooting session
