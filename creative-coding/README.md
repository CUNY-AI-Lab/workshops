# Creative Coding with Generative AI

_A CUNY AI Lab workshop bridging the history of computer art with contemporary AI-assisted creative coding in the browser._

This workshop introduces creative coding as a practice that combines rules, randomness, and interpretation. Participants explore the lineage from early computer art (Bell Labs, Vera Molnar) to today's browser-based generative workflows, then use the CUNY AI Lab Sandbox to produce single-file HTML art through structured prompts.

**Workshop Date:** Thursday, February 19, 2026
**Duration:** 90 minutes
**Format:** Interactive workshop with live demos and hands-on activities
**Facilitators:** Zach Muhlbauer & Stefano Morello (CUNY AI Lab, New Media Lab, Teaching and Learning Center)

## Overview

Through a progression from one-shot generation to iterative refinement to constrained prompting, participants learn to treat AI-generated code as raw material for curation rather than a finished product. The workshop frames the browser as a portable canvas and single-file HTML as the default deliverable.

## Learning Objectives

- Understand the historical lineage from early computer art to contemporary creative coding
- Apply the creative coding triangle (rules, randomness, interpretation) to AI-assisted workflows
- Craft structured prompts that produce runnable single-file HTML art
- Iterate on AI-generated output through conversation, constraints, and parameter tuning

## Workshop Structure

1. **Icebreaker** - Describe an algorithm you already know (conditional, loop, sequence)
2. **Origins of Computer Art** - Bell Labs (1962-1965) and early generative art
3. **Vera Molnar** - The "machine imaginaire" and rules plus disorder
4. **Creative Coding Today** - The web browser as canvas (HTML/CSS/JS)
5. **The Creative Coding Triangle** - Rules, randomness, interpretation
6. **Vibe Coding** - Prompts as prototypes (Andrej Karpathy)
7. **Behind the Scenes** - How models learn to code
8. **Coding with Intention** - Continuum from one-shot to constrained
9. **Prompt Anatomy** - Structuring prompts (role, stack, style, motion, deliverable)
10. **Live Demo** (10 min) - CUNY AI Lab Sandbox with GLM 5 / Kimi K2.5
11. **Activity 1** (10 min) - Constrained generation with system prompts
12. **Activity 2** (15 min) - Iterate and refine through conversation
13. **Activity 3** (15 min) - Parameter exploration (temperature, max tokens)
14. **Takeaways** - Save prompts + parameters + HTML output as reproducible recipes
15. **Resources** - Links, citations, and prompts
